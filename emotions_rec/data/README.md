# Data Directory

This directory holds raw and processed data for the Emotions `joy` experiment.

## raw/

The raw data directory is intentionally empty in version control. The `dair-ai/emotion` dataset is loaded programmatically via HuggingFace `datasets` in the notebook — no manual download or file placement is required.

## processed/

The processed CSVs are generated by the notebook preprocessing cells and are not tracked by Git. After running the notebook, you should find:

- `train_inner_emotions_joy.csv`
- `val_emotions_joy.csv`
- `test_emotions_joy.csv`

as well as runtime artifacts such as:

- `train_inner_emotions_joy_lda.csv`
- `train_inner_emotions_joy_data_labeled.csv`
- `train_inner_emotions_joy_training_data.csv`
- `train_inner_emotions_joy_model_results.json`
